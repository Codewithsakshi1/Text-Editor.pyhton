# Python Text Editor (Tkinter)

A simple text editor built using Python and Tkinter.  
This project demonstrates GUI development, file handling, and keyboard shortcuts.

## Features

- Dark Mode UI
- Open Text Files
- Save Files
- Scrollbar Support
- Keyboard Shortcuts
- Word & Line Counter
- Status Bar

## Technologies Used

- Python
- Tkinter

## Keyboard Shortcuts

Ctrl + N → New File  
Ctrl + O → Open File  
Ctrl + S → Save File
